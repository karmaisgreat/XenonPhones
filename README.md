
# XenonPhones - Phone Selling Web Application

**_Requirements :_**

- Python 3.7 or above
- Django 3

## Requirement Installition:-
- pip install -r requirements.txt

## Running Project:-
- python manage.py runserver

## Creating a superuser who can manage admin page:-
- python manage.py createsuperuser

## Admin login URL:-
- 127.0.0.1/admin
